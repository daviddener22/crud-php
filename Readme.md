Sobre o sistema
Esse sistema tem como objetivo cadastrar, alterar e excluir funcionários na base de dados registro.
Caso o usuário queira exportar a base, o sistema permite exportar todos os dados ordenados
pelo id da tabela em arquivo .csv.
O sistema foi desenvolvido no editor de texto SublimeText na versão 3.0, Build 3143.


Tecnologias utilizadas
- PHP 5.3.3
- HTML5
- CSS3
- Materialize 1.0.0-rc.2
- jQuery 3.3.1
